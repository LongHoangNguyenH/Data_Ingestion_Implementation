## INTRODUCTION
This project is built to evaluate and select the best machine learning model based on metrics like accuracy, F1-score, precision, etc.
some Machine Learning models used are:
+ Decision Tree
+ Random Forest
+ Gradient boosting
+ Linear Regression
+ XGBRegressor
